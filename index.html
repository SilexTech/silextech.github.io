Simple News Web App

A simple web-app created using PHP, Bootstrap and mysqli database connections. Features:

  • Everyone can browse and read the articles, as well as filter them by categories.
  • Users can register with a user/pass combination (stored in plaintext).
  • Once logged-in, Users can comment on articles and delete their own comments.
  • Admins (Users manually promoted in DB, giving them an alias) can post news and delete all news and comments.
